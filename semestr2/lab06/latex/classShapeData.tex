\doxysubsection{Detailed Description}
Abstract base class for storing serializable shape data.

Serves as the base for all shape data classes (Rectangle\+Data, Circle\+Data, Polygon\+Data). Stores common transformation and styling properties that can be saved to and loaded from a file. Implements {\ttfamily \doxylink{}{Serializable}} to support binary file I/O via {\ttfamily \doxylink{classFileIO}{File\+IO}}.
