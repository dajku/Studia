\doxysubsection{Detailed Description}
Utility class for saving and loading shape data to/from binary files.

Provides static methods for serializing an {\ttfamily \doxylink{}{Array\+List}} of {\ttfamily \doxylink{classShapeData}{Shape\+Data}} objects to a binary {\ttfamily .\+fig} file and deserializing them back. Uses Java\textquotesingle{}s built-\/in object serialization ({\ttfamily \doxylink{}{Object\+Output\+Stream}} / {\ttfamily \doxylink{}{Object\+Input\+Stream}}).
