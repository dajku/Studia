Detailed Description

Abstract base class for storing serializable shape data.

Serves as the base for all shape data classes (RectangleData, CircleData, PolygonData). Stores common transformation and styling properties that can be saved to and loaded from a file. Implements Serializable to support binary file I/O via FileIO.
