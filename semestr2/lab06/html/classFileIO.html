Detailed Description

Utility class for saving and loading shape data to/from binary files.

Provides static methods for serializing an ArrayList of ShapeData objects to a binary .fig file and deserializing them back. Uses Java's built-in object serialization (ObjectOutputStream / ObjectInputStream).
