Detailed Description

Interactive drawing canvas for creating and editing geometric shapes.

Extends javafx.scene.layout.Pane to provide a mouse-driven canvas where the user can draw, select, move, scale, rotate, and recolor shapes. Supports three shape types: circles, rectangles, and polygons.

Interaction summary:

  • Left-click + drag on empty area: draws a new shape (depending on active ToolMode).
  • Left-click on a shape: selects it (stroke thickens).
  • Left-click + drag on selected shape: moves it.
  • Scroll wheel on selected shape: scales it.
  • Shift + scroll on selected shape: rotates it.
  • Right-click on a shape: opens an inline ColorPicker to change fill color.
  • Double-click: finishes drawing a polygon.

◆ loadDrawing()

void DrawingPane.loadDrawing ( )
inline

Loads shapes from a binary file and adds them to the current canvas.

Opens a FileChooser dialog for the user to select a .fig file, then deserializes the shapes via FileIO#loadFromFile and reconstructs the corresponding JavaFX Shape objects with all saved properties restored. Loaded shapes are appended to the existing canvas without clearing it.

◆ saveDrawing()

void DrawingPane.saveDrawing ( )
inline

Saves all shapes currently on the canvas to a binary file.

Converts each Shape in the canvas into its corresponding ShapeData object (preserving all transformations and colors), then opens a FileChooser dialog for the user to choose the save location. The file is saved with a .fig extension via FileIO#saveToFile.
